# FraudVision-Sistema-Inteligente-de-Prevencao-Fraudes-Bancarias

📌 Sobre o Projeto

O FraudVision é uma siste desenvolvida para auxiliar na prevenção de golpes bancários, principalmente falsas centrais de atendimento, SMS fraudulentos e mensagens de WhatsApp.

O sistema utiliza uma IA simulada baseada em regras e palavras-chave para identificar possíveis tentativas de fraude, permitindo que usuários consultem números suspeitos, denunciem golpes e analisem mensagens recebidas.

🎯 Objetivo

Desenvolver uma solução tecnológica com foco em segurança digital e prevenção a fraudes bancárias, oferecendo uma experiência próxima de sistemas utilizados por instituições financeiras.

⚙️ Funcionalidades

* Cadastro e login de usuários
* Consulta de números suspeitos
* Denúncia de golpes
* Análise inteligente de mensagens
* Histórico de denúncias
* Classificação de risco:
    * Seguro
    * Suspeito
    * Golpe provável

* 🧠 Funcionamento da IA Simulada

A análise das mensagens funciona através de regras inteligentes e identificação de palavras-chave suspeitas, como:

* senha
* pix
* urgente
* conta bloqueada
* clique aqui

Com base nisso, o sistema gera um score de risco e alerta o usuário sobre possíveis fraudes.


📊 Diferencial do Projeto

O principal diferencial é a simulação em tempo real da detecção de golpes, proporcionando uma experiência prática e próxima de soluções corporativas utilizadas por bancos.
