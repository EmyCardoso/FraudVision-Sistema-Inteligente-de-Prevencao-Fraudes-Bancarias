Funcionamento da IA Simulada

A análise das mensagens funciona através de regras inteligentes e identificação de palavras-chave suspeitas, como:

- senha  
- pix  
- urgente  
- conta bloqueada  
- clique aqui  

Com base nisso, o sistema gera um score de risco e alerta o usuário sobre possíveis fraudes.

 📊 Diferencial do Projeto

O principal diferencial é a simulação em tempo real da detecção de golpes, proporcionando uma experiência prática e próxima de soluções corporativas utilizadas por bancos.
